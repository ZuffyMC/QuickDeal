# QuickDeal Support

Need help with QuickDeal?

Please create a GitHub Issue and include:

- QuickDeal version
- Server version
- Java version
- Plugin list
- Economy plugin
- Storage type
- Error logs
- Steps to reproduce

## Common Checks

1. Vault is installed.
2. A real economy plugin is connected to Vault.
3. Config files are not broken.
4. GUI files are saved as UTF-8.
5. You are using Java 21+.
6. You are using Paper or Spigot 1.21+.
