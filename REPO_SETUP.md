# QuickDeal GitHub Repo Setup

1. Create a new GitHub repository named `QuickDeal`.
2. Upload all files from this folder.
3. Make sure `assets/banner.png` exists.
4. Set repository description:

```txt
Modern AuctionHouse plugin with marketplace GUI, drag-and-drop selling, seller ratings, player profiles, auto backup, transaction logs, and full customization.
```

5. Suggested topics:

```txt
minecraft-plugin
auctionhouse
auction-house
paper-plugin
spigot-plugin
minecraft-economy
minecraft-marketplace
vault
gui-plugin
quickdeal
```

6. Add screenshots later into `assets/screenshots/`.
