# Security Policy

If you find a security issue, duplication bug, item loss bug, or economy exploit, please report it through GitHub Issues with as much detail as possible.

Please include:

- QuickDeal version
- Server version
- Java version
- Economy plugin
- Storage type
- Steps to reproduce
- Screenshots or logs if available

QuickDeal security focuses on anti-dupe protection, safe item return, transaction validation, economy validation, inventory full protection, pending claim safety, and listing state validation.
