# QuickDeal Commands

## Player Commands

| Command | Description |
|---|---|
| `/ah` | Open QuickDeal dashboard |
| `/ah browse` | Open marketplace browser |
| `/ah sell` | Open drag & drop sell GUI |
| `/ah sell <price>` | Sell item in hand with price |
| `/ah listings` | View your active listings |
| `/ah expired` | Claim expired items |
| `/ah sold` | View sold items |
| `/ah history` | View trading history |
| `/ah profile` | View your market profile |
| `/ah profile <player>` | View another player's profile |
| `/ah leaderboard` | View top sellers |
| `/ah rating` | Open rating menu |
| `/ah claims` | Open pending claims |
| `/ah help` | Open help center |

Aliases: `/quickdeal`, `/auctionhouse`, `/auction`, `/qd`, `/deal`

## Admin Commands

| Command | Description |
|---|---|
| `/qda` | Open admin dashboard |
| `/qda reload` | Reload config, GUI, messages, sounds, and categories |
| `/qda backup create` | Create manual backup |
| `/qda backup list` | View backups |
| `/qda backup restore <file>` | Restore backup |
| `/qda listings` | View all listings |
| `/qda listings <player>` | View player listings |
| `/qda remove <listingId>` | Remove listing |
| `/qda logs` | View transaction logs |
| `/qda logs <player>` | View player transaction logs |
| `/qda debug` | View debug information |
| `/qda storage status` | View storage status |
| `/qda economy status` | View economy provider status |
| `/qda testdata <amount>` | Generate test listings |

Aliases: `/quickdealadmin`, `/ahadmin`
