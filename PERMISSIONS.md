# QuickDeal Permissions

## Player Permissions

| Permission | Description | Default |
|---|---|---|
| `quickdeal.use` | Access QuickDeal | true |
| `quickdeal.open` | Open dashboard | true |
| `quickdeal.sell` | Sell items | true |
| `quickdeal.buy` | Buy items | true |
| `quickdeal.cancel` | Cancel own listings | true |
| `quickdeal.history` | View history | true |
| `quickdeal.profile` | View profiles | true |
| `quickdeal.rating` | Rate sellers | true |
| `quickdeal.leaderboard` | View leaderboard | true |
| `quickdeal.claims` | Open pending claims | true |

## Admin Permissions

| Permission | Description |
|---|---|
| `quickdeal.admin` | Access admin tools |
| `quickdeal.admin.reload` | Reload plugin files |
| `quickdeal.admin.backup` | Manage backups |
| `quickdeal.admin.restore` | Restore backups |
| `quickdeal.admin.remove` | Remove listings |
| `quickdeal.admin.logs` | View logs |
| `quickdeal.admin.debug` | Use debug tools |
| `quickdeal.admin.testdata` | Generate test data |

## Limit Permissions

```txt
quickdeal.limit.default
quickdeal.limit.vip
quickdeal.limit.elite
quickdeal.limit.unlimited
```
